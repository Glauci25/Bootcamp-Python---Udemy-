# Bootcamp-Python---Udemy-
Trata-se de um Bootcamp de Python de 100 dias de aprendizado, sendo desenvolvido um projeto por dia. 

A dificuldade dos projetos evolui conforme os dias de aprendizado, indo de iniciante ao avançado. Talvez nem todos os dias estejam presentes nas pastas, isso porque alguns podem ter se perdido no meio do caminho, mas é a vida.
